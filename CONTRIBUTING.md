# Contributing to RootRouter

Thank you for your interest in contributing to RootRouter.

## How to contribute

- **Bug reports & feature ideas:** Open an [issue](https://github.com/Motus-DAO/RootRouter/issues).
- **Code & docs:** Open a pull request against `main`. Keep changes focused and include a short description.

## Development setup

```bash
git clone https://github.com/Motus-DAO/RootRouter.git
cd RootRouter
npm install
npm run build
```

Run demos: `npx tsx demo/basic.ts`, `npx tsx demo/benchmark.ts`, `npx tsx demo/swarm.ts`.

## Code style

- TypeScript; follow existing patterns in `src/`.
- Run `npm run build` before submitting; the project has no extra lint config beyond the build.
